Les bases de SASS et SCSS

  • pour instaler SASS, on utilisera NPM.
    npm install -g sass la commande pour complier du sass/scss en css est
    sass nomDuFichier.scss si la commande n'est pas reconnu, tentez de mettre "npx" avant.
  • pour eviter de rataper la commande à chaque fois on peut utiliser l'option "--watch" pour que le terminal reste en surveillance sass --watch nomDuFichier.scss nomDuFichier.css
  • Pour surveiller tout un dossier: sass --watch nomDuFichierSource/:nomDuDossierCible/
  • On peut minimiser directement le CSS avec: sass --watch nomDuDossierSource/::nomDuDossierCible/ --style compressed
